## 🛠️ Tecnologías Utilizadas

El portafolio está construido con tecnologías modernas de desarrollo web:

- **HTML5**: Estructura semántica del sitio.
- **CSS3 (Custom Properties & Flexbox/Grid)**: Diseño visual, animaciones y respuesta a dispositivos móviles.
- **JavaScript (Vanilla)**: Lógica para efectos visuales y navegación.
- **Google Fonts**: Tipografías _Syne_ y _DM Sans_ para una estética premium.

## 💻 Cómo verlo localmente

Si deseas clonar el repositorio y ejecutarlo en tu máquina local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/MauriiLogx/Portafolio-Personal.git
    ```
2.  **Entrar al directorio:**
    ```bash
    cd Portafolio-Personal
    ```
3.  **Abrir el archivo principal:**
    Simplemente abre el archivo `index.html` en tu navegador preferido.
